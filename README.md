1.	–°–ø—Ä–æ–µ–∫—Ç–∏—Ä—É–π—Ç–µ –∏ —Ä–µ–∞–ª–∏–∑—É–π—Ç–µ —Å–µ—Ä–≤–∏—Å, –ø–æ–∑–≤–æ–ª—è—é—â–∏–π –≤—ã—á–∏—Å–ª—è—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è —Å–∏–º–≤–æ–ª–æ–≤ –õ–µ–∂–∞–Ω–¥—Ä–∞ –∏ –Ø–∫–æ–±–∏.
2.	–°–ø—Ä–æ–µ–∫—Ç–∏—Ä—É–π—Ç–µ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å, –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—é—â–∏–π –æ–ø–∏—Å–∞–Ω–∏–µ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª–∞ –¥–ª—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–Ω–æ–≥–æ —Ç–µ—Å—Ç–∞ –ø—Ä–æ—Å—Ç–æ—Ç—ã (–ø–∞—Ä–∞–º–µ—Ç—Ä—ã –º–µ—Ç–æ–¥–∞: —Ç–µ—Å—Ç–∏—Ä—É–µ–º–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –º–∏–Ω–∏–º–∞–ª—å–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø—Ä–æ—Å—Ç–æ—Ç—ã –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ [0.5, 1) ). –° –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º —Å–µ—Ä–≤–∏—Å–∞, —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–Ω–æ–≥–æ –≤ –∑–∞–¥–∞–Ω–∏–∏ 1, —Ä–µ–∞–ª–∏–∑—É–π—Ç–µ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å –¥–ª—è —Å–ª–µ–¥—É—é—â–∏—Ö –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–Ω—ã—Ö —Ç–µ—Å—Ç–æ–≤ –ø—Ä–æ—Å—Ç–æ—Ç—ã: –§–µ—Ä–º–∞, –°–æ–ª–æ–≤–µ—è-–®—Ç—Ä–∞—Å—Å–µ–Ω–∞, –ú–∏–ª–ª–µ—Ä–∞-–†–∞–±–∏–Ω–∞.
3.	–°–ø—Ä–æ–µ–∫—Ç–∏—Ä—É–π—Ç–µ –∏ —Ä–µ–∞–ª–∏–∑—É–π—Ç–µ —Å–µ—Ä–≤–∏—Å, –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω–Ω—ã–π –¥–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è —à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è –∏ –¥–µ—à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö –∞–ª–≥–æ—Ä–∏—Ç–º–æ–º RSA. –°–µ—Ä–≤–∏—Å –¥–æ–ª–∂–µ–Ω —Å–æ–¥–µ—Ä–∂–∞—Ç—å –æ–±—ä–µ–∫—Ç –≤–ª–æ–∂–µ–Ω–Ω–æ–≥–æ —Å–µ—Ä–≤–∏—Å–∞ –¥–ª—è –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –∫–ª—é—á–µ–π –∞–ª–≥–æ—Ä–∏—Ç–º–∞ RSA (–∫–æ–Ω—Ç—Ä–∞–∫—Ç –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä–∞ –≤–ª–æ–∂–µ–Ω–Ω–æ–≥–æ —Å–µ—Ä–≤–∏—Å–∞: –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–π —Ç–µ—Å—Ç –ø—Ä–æ—Å—Ç–æ—Ç—ã (–∑–∞–¥–∞—ë—Ç—Å—è –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω–∏–µ–º), –º–∏–Ω–∏–º–∞–ª—å–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø—Ä–æ—Å—Ç–æ—Ç—ã –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ [0.5, 1), –±–∏—Ç–æ–≤–∞—è –¥–ª–∏–Ω–∞ —Å–≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –ø—Ä–æ—Å—Ç—ã—Ö —á–∏—Å–µ–ª; –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –¥–µ–ª–µ–≥–∏—Ä—É—é—Ç—Å—è –∏–∑ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä–∞ —Å–µ—Ä–≤–∏—Å–∞-–æ–±—ë—Ä—Ç–∫–∏). –ü—Ä–∏ –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –∫–ª—é—á–µ–π –æ–±–µ—Å–ø–µ—á—å—Ç–µ –∑–∞—â–∏—Ç—É –æ—Ç –∞—Ç–∞–∫–∏ –§–µ—Ä–º–∞ –∏ –∞—Ç–∞–∫–∏ –í–∏–Ω–µ—Ä–∞. –ü—Ä–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–∏ –æ–ø–µ—Ä–∞—Ü–∏–π —à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è –æ–±–µ—Å–ø–µ—á—å—Ç–µ –∑–∞—â–∏—Ç—É –æ—Ç –∞—Ç–∞–∫–∏ –•–∞—Å—Ç–∞–¥–∞. –ù–æ–≤—É—é –∫–ª—é—á–µ–≤—É—é –ø–∞—Ä—É –º–æ–∂–Ω–æ –≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞—Ç—å –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–∞–∑. –ü—Ä–æ–¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É–π—Ç–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ —à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è –∏ –¥–µ—à–∏—Ñ—Ä–æ–≤–∞–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö –∞–ª–≥–æ—Ä–∏—Ç–º–æ–º RSA –ø–æ—Å—Ä–µ–¥—Å—Ç–≤–æ–º —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–Ω–æ–≥–æ —Å–µ—Ä–≤–∏—Å–∞.
4.	–†–µ–∞–ª–∏–∑—É–π—Ç–µ —Å–µ—Ä–≤–∏—Å, –¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É—é—â–∏–π –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –∞—Ç–∞–∫–∏ –í–∏–Ω–µ—Ä–∞ –Ω–∞ –æ—Ç–∫—Ä—ã—Ç—ã–π –∫–ª—é—á –∞–ª–≥–æ—Ä–∏—Ç–º–∞ RSA. –í –∫–∞—á–µ—Å—Ç–≤–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø–æ–ª—É—á–∏—Ç—å –∫–æ–ª–ª–µ–∫—Ü–∏—é –ø–æ–¥—Ö–æ–¥—è—â–∏—Ö –¥—Ä–æ–±–µ–π –¥–ª—è —Ü–µ–ø–Ω–æ–π –¥—Ä–æ–±–∏ ùëÅùëí –∏ –Ω–∞–π–¥–µ–Ω–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –¥–µ—à–∏—Ñ—Ä—É—é—â–µ–π —ç–∫—Å–ø–æ–Ω–µ–Ω—Ç—ã.
